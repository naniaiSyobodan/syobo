集合・整列
  1. 「集まれ」
  2. 基準員が班長正面から５ｍに移動し「基準」
  3. 他の団員は基準員の左に並び、自主整頓
  4. 団員の列の中央に移動し「右へならえ」
  5. 「直れ」
  6. 「番号」
  7. 「イチ・ニ・サン・シ・ゴ・ロク・シチ・ハチ・・」
  8. 基準員の右から１ｍの位置に移動

  1. 「巡視者にカシラ中」
  2. 挙手の敬礼
  3. 巡視者に顔を向ける
  4. 巡視者が手を降ろしたら「直れ」
  5. 敬礼ヤメ
  6. 正面を向く
報告
  1. 巡視者に向かって挙手の敬礼
  2. 巡視者が手を下ろしたら敬礼ヤメ
  3. 「長野市七二会分団〇〇班

  6. 挙手の敬礼
  7. 巡視者が手を下ろしたら敬礼ヤメ
訓示・激励
  1. 巡視者の指示があった場合「整列休め」
  2. 巡視者の話が終わったら「気をつけ」
  3. 「カシラ中」挙手の敬礼
  4. 巡視者が手を降ろしたら「直れ」

  2. 「別れ」挙手の敬礼

  4. 敬礼ヤメ
